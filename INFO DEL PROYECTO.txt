│   ├── translation/

│   ├── subtitles/
│   │   ├── subtitle_generator.py   # Genera subtítulos .srt / .vtt

│   │
│   ├── pipeline/
│   │   ├── media_processor.py      # Orquestador: audio → texto → traducción → subtítulos
│   │
│   ├── utils/
│   │   ├── file_handlers.py        # Manejo de archivos MP3, WAV, MP4, SRT, etc.

│   │
│   ├── app.py                      # Interfaz principal (CLI o API Flask)

├── models/

├── README.md                       # Documentación del proyecto

| Tipo                  | Formato             | Descripción              |
| --------------------- | ------------------- | ------------------------ |
| **Audio sin pérdida** | WAV, AIFF, FLAC     | Calidad alta             |
| **Audio comprimido**  | MP3, AAC, OGG, Opus | Más ligero               |
| **Video contenedor**  | MP4, MKV, MOV       | Guarda video + audio     |
| **Códec de video**    | H.264, H.265, AV1   | Forma en que se comprime |
| **Códec de audio**    | AAC, MP3, Opus      | Comprime el audio        |
